# 🚁 Drone Store – Smart E-Commerce Platform

## Overview
Drone Store is a MERN stack e-commerce platform focused on selling drones and drone accessories.  
It includes an AI-powered recommendation system that helps users find the best drone based on their needs instead of browsing randomly.

---

## Core Idea
Help users choose the right drone by asking simple questions (purpose, budget, experience) and recommending the most suitable products with clear reasoning.

---

## Key Features

### User
- Authentication (Login / Signup)
- Browse drones & accessories
- AI-based drone finder
- Product comparison
- Cart & checkout
- Wishlist
- Order history
- Reviews & ratings

### AI Recommendation
- Purpose-based recommendations (Photography, Racing, Survey, Hobby)
- Budget-aware suggestions
- Beginner / Pro friendly filtering
- “Why this drone?” explanation
- Similar & alternative drone suggestions

### Admin
- Admin login
- Add / update / delete products
- Inventory management
- Order management
- User management

---

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Auth: JWT/
- AI: Rule-based recommendation system (ML-ready)
- Cloud: Cloudinary
- Deployment: Vercel, Render
